---
title: $isNumber
description: Checks if the value is a number or not.
---

This function checks if the &lt;message&gt; is a number or not. Returns boolean

## Usage

```php
$isNumber[text]
```

### Example

```javascript
bot.command({
name: "isnumber", 
code: `
$isNumber[12432]` //This will return true
})
```
