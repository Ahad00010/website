---
title: $thumbnail
description: Adds a thumbnail to the embed
---

This function adds a thumbnail to the embed

```php
$thumbnail[index;url]
```

```javascript
bot.command({
name: "thumbnail",
code: `$thumbnail[1;https://cdn.discordapp.com/avatars/746401637329010779/a3dc97600375b95156a33d0fccbf2c95.webp]`
})
```
