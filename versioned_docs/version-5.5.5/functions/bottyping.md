---
title: $botTyping
description: Displays the bot as typing
---

This function just displays the bot as 'Typing'.

### Usage 
```php
$botTyping
```


## Examples

```javascript
bot.command({
name: "bot",
code: `I was just typing
$botTyping`
})
```
