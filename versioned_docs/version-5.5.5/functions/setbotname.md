---
title: $setBotName
description: Changes the bot's name to the given text
---

This function set's the bot's username

```php
$setBotName[text]
```

```javascript
bot.command({
name: "setbotname",
code: `Set the bots name! 
$setBotName[Aoi.JS]`
})
```

> THIS SHOULD BE RESTRICTED TO BOT DEV ONLY
