---
title: $isMuted
description: Checks if given user ID is muted. Returns true/false
---

This function checks if the given user ID is muted. Returns boolean

## Usage

```php
$isMuted[userID]
```

### Example

```javascript
bot.command({
name: "isMuted",
code: `Is Muted: $isMuted[535566311942651924]`
})
```
