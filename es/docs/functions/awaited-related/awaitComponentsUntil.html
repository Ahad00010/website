$awaitComponentsUntil

$awaitComponentsUntil espera los componentes del mensaje.

Uso​

$awaitComponentsUntil[CanalID;ID de mensaje;filtro de usuario;tiempo;ID personalizados;comandos;mensaje de error?;esperar datos?]

Parámetros​

CampoTipoDescripciónRequerido
canalIDcadenaID del canal.verdadero
ID de mensajecadenaMensaje ID.verdadero
filtro de usuarioenteroA lo que el bot responderá
1. todos
2. ID de usuario específico - cualquier ID de usuario		verdadero
reaccionescadenaReacciones que escuchará el bot, puede separar varios emojis con una coma ( , )verdadero
comandoscadenaComandos que se ejecutarán, puedes separar múltiples emojis con una coma (,)verdadero
mensaje de error?cadenaMensaje de error cuando caduca el comando.falso
esperar datos?cadenaDatos esperados.falso
