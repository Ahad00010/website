`$oldEmoji` зберігає дані для емодзі перед оновленням/видаленням. (події emojiUpdate та emojiDelete подія)

## Використання

```php
$oldEmoji[option]
```

## Параметри

| Поле  | Тип   | Опис                 | Обов'язковий |
| ----- | ----- | -------------------- |:------------:|
| опція | рядок | Можливість отримати. |     так      |
