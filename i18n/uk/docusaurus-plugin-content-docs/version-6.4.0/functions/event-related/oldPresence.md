`$oldPresence` зберігає старі дані присутності користувача, що використовуються в оновленні присутності події, або будь-які дані будуть пусті

## Використання

```php
$oldPresence[option]
```

## Параметри

| Поле  | Тип   | Опис                 | Обов'язковий |
| ----- | ----- | -------------------- |:------------:|
| опція | рядок | Можливість отримати. |     так      |
