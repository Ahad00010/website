`$oldRole` зберігає дані ролі до його оновлення, використовуються в події оновлення ролей або будь-які дані будуть порожніми.

## Використання

```php
$oldRole[option]
```

## Параметри

| Поле  | Тип   | Опис                 | Обов'язковий |
| ----- | ----- | -------------------- |:------------:|
| опція | рядок | Можливість отримати. |     так      |
