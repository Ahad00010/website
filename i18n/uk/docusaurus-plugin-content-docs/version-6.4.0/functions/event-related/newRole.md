`$newRole` holds data of a role that was created, updated, or deleted. Used in role update / delete / create events, or else it won't give any data.

## Використання

```php
$newRole[option]
```

| Поле   | Тип   | Опис                | Обов'язковий |
| ------ | ----- | ------------------- |:------------:|
