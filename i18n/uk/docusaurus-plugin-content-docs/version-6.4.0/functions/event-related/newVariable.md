`$newVariable` отримує дані з подій змінної.

## Використання

```php
$newVariable[option;sep?]
```

## Параметри

| Поле  | Тип   | Опис                 | Обов'язковий |
| ----- | ----- | -------------------- |:------------:|
| опція | рядок | Можливість отримати. |     так      |
| вер   | рядок | Роздільник.          |      ні      |
