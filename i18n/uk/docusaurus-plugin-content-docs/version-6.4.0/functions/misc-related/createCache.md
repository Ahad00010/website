`$createCache` will create custom cache.

## Використання

```php
$createCache[type;name;options]
```

## Параметри

| Поле    | Тип   | Опис           | Обов'язковий |
| ------- | ----- | -------------- |:------------:|
| тип     | рядок | Cache type.    |     так      |
| назва   | рядок | Cache name.    |     так      |
| options | рядок | Cache options. |     так      |
