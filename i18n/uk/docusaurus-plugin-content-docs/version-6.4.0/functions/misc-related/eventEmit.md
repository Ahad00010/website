`$eventEmit` випромінить настроювану подію.

## Використання

```php
$eventEmit[name;...datas]
```

## Параметри

| Поле     | Тип   | Опис                 | Обов'язковий |
| -------- | ----- | -------------------- |:------------:|
| назва    | рядок | Власна назва події.  |     так      |
| ...datas | рядок | Довільні дані подій. |     так      |
