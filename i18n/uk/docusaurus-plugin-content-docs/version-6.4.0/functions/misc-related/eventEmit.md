`$eventEmit` will emit a custom event.

## Використання

```php
$eventEmit[name;...datas]
```

## Параметри

| Поле     | Тип   | Опис               | Обов'язковий |
| -------- | ----- | ------------------ |:------------:|
| назва    | рядок | Custom event name. |     так      |
| ...datas | рядок | Custom event data. |     так      |
