`$ram` поверне використання ОЗУ вашого бота.

## Використання

```php
$ram
```

## Приклад(и)

Це поверне використання оперативної пам'яті та кількість оперативної пам'яті:

```javascript
bot.command({
    name: 'ram',
    код: `
  я використовую \`$ram%\` з \`$maxRam%\`!
  `
});
```
