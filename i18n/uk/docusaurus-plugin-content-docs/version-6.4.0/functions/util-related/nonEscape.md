`$nonEscape` перестане екранувати спеціальні символи.

## Використання

```php
$nonEscape[message]
```

## Параметри

| Поле         | Тип   | Опис                                               | Обов'язковий |
| ------------ | ----- | -------------------------------------------------- | ------------ |
| повідомлення | рядок | Текст, який ви не хочете, щоб вас використовували. | так          |

## Приклад(и)

Це перестане уникати певних символів:

```javascript
bot.command({
    name: 'nonEscape',
    код: `
  $nonEscape[Hello [;)]]
  `
});
```
