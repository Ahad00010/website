| Поле       | Тип   | Опис                  | Обов'язковий |
| ---------- | ----- | --------------------- |:------------:|
| текст      | рядок | Текст для приєднання. |     так      |
| розділювач | рядок | Як розділитися.       |     так      |
| sep1       | рядок | Роздільник.           |      ні      |
| sep2       | рядок | Розділювач.           |      ні      |
