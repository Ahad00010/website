`$newState` contiene datos para el estado de voz del usuario después de la actualización (evento voiceStateUpdate).

## Uso

```php
$newState[opción]
```

## Parámetros

| Campo  | Tipo   | Descripción         | Requerido |
| ------ | ------ | ------------------- |:---------:|
| opción | cadena | Opción a recuperar. | verdadero |
