`$oldMsgData` contiene datos del mensaje anterior.

## Uso

```php
$oldMsgData[opción]
```

## Parámetros

| Campo  | Tipo   | Descripción         | Requerido |
| ------ | ------ | ------------------- |:---------:|
| opción | cadena | Opción a recuperar. | verdadero |
