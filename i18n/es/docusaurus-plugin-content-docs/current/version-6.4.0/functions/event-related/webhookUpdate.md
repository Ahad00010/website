Un canal `$webhookUpdate` en el que se actualizó el webhook. (evento webhookUpdate )

## Uso

```php
$webhookUpdate[opción]
```

## Parámetros

| Campo  | Tipo   | Descripción         | Requerido |
| ------ | ------ | ------------------- |:---------:|
| opción | cadena | Opción a recuperar. | verdadero |
