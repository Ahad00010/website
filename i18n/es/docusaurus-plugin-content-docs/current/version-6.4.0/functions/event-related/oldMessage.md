`$oldMessage` contiene el contenido del mensaje antes de que se actualizara. (si hay alguna)

## Uso

```php
$oldMessage[opción]
```

## Parámetros

| Campo  | Tipo   | Descripción         | Requerido |
| ------ | ------ | ------------------- |:---------:|
| opción | cadena | Opción a recuperar. | verdadero |
