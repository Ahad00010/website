`$createCache` will create custom cache.

## Modo de uso

```php
$createCache[type;name;options]
```

## Parámetros

| Campo   | Tipo     | Parámetros     | Requerido |
| ------- | -------- | -------------- |:---------:|
| type    | consulta | Cache type.    | verdadero |
| name    | consulta | Cache name.    | verdadero |
| options | string   | Cache options. |    sí     |
