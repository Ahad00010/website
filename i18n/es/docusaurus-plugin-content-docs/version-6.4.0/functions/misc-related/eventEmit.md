`$eventEmit` will emit a custom event.

## Modo de uso

```php
$eventEmit[name;...datas]
```

## Parámetros

| Campo    | Tipo     | Parámetros         | Requerido |
| -------- | -------- | ------------------ |:---------:|
| nombre   | consulta | Custom event name. | verdadero |
| ...datas | consulta | Custom event data. | verdadero |
