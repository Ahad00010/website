`$lerefAvatar` devolverá el Avatar del creador de aoi.js

## Uso

```php
$lerefAvatar
```

## Ejemplo(s)

Esto devolverá el Avatar del creador:

```javascript
bot.command({
    name: "lerefAvatar",
    code: `
    $lerefAvatar
    `
});
```
