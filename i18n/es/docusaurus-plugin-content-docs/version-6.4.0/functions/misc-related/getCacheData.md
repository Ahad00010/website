`$getCacheData` devolverá los datos del caché.

## Uso

```php
$getCacheData[tipo;nombre;llave;opcion?]
```

## Parámetros

| Campo   | Tipo   | Parámetros | Requerido |
| ------- | ------ | ---------- |:---------:|
| tipo    | cadena |            | verdadero |
| nombre  | cadena |            | verdadero |
| llave   | cadena |            | verdadero |
| opción? | cadena |            |   falso   |
