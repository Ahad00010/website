`$getCacheData` will return cache data.

## Uso

```php
$getCacheData[type;name;key;option?]
```

## Parámetros

| Campo   | Tipo     | Parámetros | Requerido |
| ------- | -------- | ---------- |:---------:|
| type    | consulta |            | verdadero |
| name    | consulta |            | verdadero |
| key     | string   |            |    sí     |
| opción? | string   |            |   falso   |
