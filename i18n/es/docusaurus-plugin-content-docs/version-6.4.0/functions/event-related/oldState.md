`$oldState` contiene datos para el estado de voz del usuario antes de que se actualizara (evento voiceStateUpdate).

## Uso

```php
$oldState[opción]
```

## Parámetros

| Campo  | Tipo   | Descripción         | Requerido |
| ------ | ------ | ------------------- |:---------:|
| opción | cadena | Opción a recuperar. | verdadero |
