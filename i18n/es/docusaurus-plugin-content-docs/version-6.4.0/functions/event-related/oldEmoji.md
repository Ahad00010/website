`$oldEmoji` contiene datos para el emoji antes de que se actualizara/eliminara. (emojiUpdate and emojiDelete evento)

## Uso

```php
$oldEmoji[opción]
```

## Parámetros

| Campo  | Tipo   | Descripción         | Requerido |
| ------ | ------ | ------------------- |:---------:|
| opción | cadena | Opción a recuperar. | verdadero |
