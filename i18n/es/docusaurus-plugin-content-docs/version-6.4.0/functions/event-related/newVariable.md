`$newVariable` obtiene los datos de eventos variables.

## Uso

```php
$newVariable[opción;sep?]
```

## Parámetros

| Campo  | Tipo   | Descripción         | Requerido |
| ------ | ------ | ------------------- |:---------:|
| opción | cadena | Opción a recuperar. | verdadero |
| sep    | cadena | Separador           |   falso   |
