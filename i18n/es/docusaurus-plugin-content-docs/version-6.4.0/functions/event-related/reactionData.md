`$reactionData` contiene datos de una reacción.

## Uso

```php
$reactionData[opción]
```

## Parámetros

| Campo  | Tipo   | Descripción                                                                                            | Requerido |
| ------ | ------ | ------------------------------------------------------------------------------------------------------ |:---------:|
| opción | cadena | Opción a recuperar. <br /> 1. **usernames** <br /> 2. **userIds** <br /> 3. **tags** | verdadera |
