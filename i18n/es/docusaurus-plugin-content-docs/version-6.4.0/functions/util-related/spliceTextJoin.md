`$spliceTextJoin`

## Modo de uso

```php
$spliceTextJoin[text;splitter;sep1?;sep2?]
```

## Parámetros

| Campo    | Tipo     | Parámetros    | Requerido |
| -------- | -------- | ------------- |:---------:|
| text     | consulta | Text to join. | verdadero |
| splitter | consulta | How to split. | verdadero |
| sep1     | string   | Separador.    |   falso   |
| sep2     | string   | Separador.    |   falso   |
