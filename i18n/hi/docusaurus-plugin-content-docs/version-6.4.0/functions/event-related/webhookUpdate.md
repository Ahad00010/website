`$webhookUpdate` channel the webhook was updated in. (webhookUpdate event)

## प्रोयोग

```php
$webhookUpdate[option]
```

## पैरामीटर्स

| फील्ड  | टाइप     | डिस्क्रिप्शन        | चाहिए |
| ------ | -------- | ------------------- |:-----:|
| option | स्ट्रिंग | Option to retrieve. | true  |
