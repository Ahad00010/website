`$oldMessage` holds the content of the message before it was updated. (if any)

## प्रोयोग

```php
$oldMessage[option]
```

## पैरामीटर्स

| फील्ड  | टाइप     | डिस्क्रिप्शन        | चाहिए |
| ------ | -------- | ------------------- |:-----:|
| option | स्ट्रिंग | Option to retrieve. | true  |
