`$newVariable` gets the data from variable events.

## प्रोयोग

```php
$newVariable[option;sep?]
```

## पैरामीटर्स

| फील्ड  | टाइप     | डिस्क्रिप्शन        |    चाहिए     |
| ------ | -------- | ------------------- |:------------:|
| option | स्ट्रिंग | Option to retrieve. |     true     |
| sep    | स्ट्रिंग | Separator.          | असत्य (नहीं) |
