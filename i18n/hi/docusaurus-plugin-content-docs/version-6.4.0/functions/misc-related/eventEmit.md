`$eventEmit` will emit a custom event.

## प्रोयोग

```php
$eventEmit[name;...datas]
```

## पैरामीटर्स

| फील्ड    | टाइप     | डिस्क्रिप्शन       | चाहिए |
| -------- | -------- | ------------------ |:-----:|
| name     | स्ट्रिंग | Custom event name. | true  |
| ...datas | स्ट्रिंग | Custom event data. | true  |
