`$createCache` will create custom cache.

## प्रोयोग

```php
$createCache[type;name;options]
```

## पैरामीटर्स

| फील्ड   | टाइप     | डिस्क्रिप्शन   | चाहिए |
| ------- | -------- | -------------- |:-----:|
| टाइप    | स्ट्रिंग | Cache type.    | true  |
| name    | स्ट्रिंग | Cache name.    | true  |
| options | स्ट्रिंग | Cache options. | true  |
