`$getCacheData` will return cache data.

## प्रोयोग

```php
$getCacheData[type;name;key;option?]
```

## पैरामीटर्स

| फील्ड   | टाइप     | डिस्क्रिप्शन |    चाहिए     |
| ------- | -------- | ------------ |:------------:|
| टाइप    | स्ट्रिंग |              |     true     |
| name    | स्ट्रिंग |              |     true     |
| key     | स्ट्रिंग |              |     true     |
| option? | स्ट्रिंग |              | असत्य (नहीं) |
