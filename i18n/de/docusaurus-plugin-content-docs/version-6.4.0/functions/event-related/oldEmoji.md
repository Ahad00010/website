`$oldEmoji` enthält Daten für das Emoji, bevor es aktualisiert wurde / gelöscht wurde. (EmojiUpdate und EmojiDelete Ereignis)

## Verwendung

```php
$oldEmoji[option]
```

## Verwendung

| Feld      | Typ    | Beschreibung        | Erforderlich |
| --------- | ------ | ------------------- |:------------:|
| variieren | string | Option zum Abrufen. |     wahr     |
