`$newVariable` holt die Daten von variablen Ereignissen.

## Verwendung

```php
$newVariable[option;sep?]
```

## Verwendung

| Feld      | Typ    | Beschreibung        | Erforderlich |
| --------- | ------ | ------------------- |:------------:|
| variieren | string | Option zum Abrufen. |     wahr     |
| sep       | string | Trennzeichen.       |    falsch    |
