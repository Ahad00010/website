`$oldState` speichert Daten für den Sprachstatus des Benutzers, bevor er aktualisiert wurde (VoiceStateUpdate Ereignis).

## Verwendung

```php
$oldState[option]
```

## Verwendung

| Feld      | Typ    | Beschreibung        | Erforderlich |
| --------- | ------ | ------------------- |:------------:|
| variieren | string | Option zum Abrufen. |     wahr     |
