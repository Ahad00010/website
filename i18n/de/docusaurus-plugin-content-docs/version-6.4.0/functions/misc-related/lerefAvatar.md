`$lerefAvatar` wird den Avatar des Erstellers von aoi.js zurückgeben

## Verwendung

```php
$lerefAvatar
```

## Beispiel(e)

Dadurch wird der Ersteller Avatar zurückgegeben:

```javascript
bot.command({
    name: "lerefAvatar",
    code: `
    $lerefAvatar
    `
});
```
