`$eventEmit` wird ein benutzerdefiniertes Ereignis abgeben.

## Verwendung

```php
$eventEmit[name;...datas]
```

## Verwendung

| Feld     | Typ    | Beschreibung          | Erforderlich |
| -------- | ------ | --------------------- |:------------:|
| Name     | string | Eigener Ereignisname. |     wahr     |
| ...datas | string | Eigene Ereignisdaten. |     wahr     |
