`$arrayConcat` werden mehrere Arrays konnektieren.

## Verwendung

```php
$arrayConcat[trennzeichen;...arrays]
```

## Parameter

| Feld         | Typ    | Beschreibungen   | Pflichtfeld |
| ------------ | ------ | ---------------- |:-----------:|
| trennzeichen | string | Trennzeichen.    |    wahr     |
| ...arrays    | string | Name des Arrays. |    wahr     |
