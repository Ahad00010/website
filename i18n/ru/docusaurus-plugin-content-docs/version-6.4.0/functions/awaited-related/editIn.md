`$editIn` отредактирует сообщение через заданное время.

## Использование

```php
$editIn[time;...content]
```

## Параметры

| Название | Nbg           | Описание                                               | Нужно |
| -------- | ------------- | ------------------------------------------------------ |:-----:|
| время    | строка, число | После сколько времени будет отредактировано сообщение. |  да   |
| контент  | строка        | Какое новое содержимое должно быть сообщение.          |  да   |

## Пример(ы)

Это отредактирует отправленное сообщение через пять секунд:

```javascript
bot.command({
    name: 'editIn',
    code: `
  $editIn[5s;aoi.js is great, don't you agree?;Yes, I do!]
  Я буду редактировать это сообщение через 5 секунд!
  «
});
```
