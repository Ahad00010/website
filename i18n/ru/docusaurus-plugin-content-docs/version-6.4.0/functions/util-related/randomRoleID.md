`$randomRoleID` будет возвращать случайный ID роли данной гильдии.

## Использование

```php
$randomRoleID[guildID?]
```

## Параметры

| Название    | Nbg   | Описание                                                | Нужно |
| ----------- | ----- | ------------------------------------------------------- |:-----:|
| ID гильдии? | целое | Где он будет возвращать роль случайного идентификатора. |  да   |

## Пример(ы)

Будет возвращена случайный ID вашей гильдии:

```javascript
bot.command({
    name: 'randomRoleID',
    code: `
  $randomRoleID[$guildID]
  `
});
```
