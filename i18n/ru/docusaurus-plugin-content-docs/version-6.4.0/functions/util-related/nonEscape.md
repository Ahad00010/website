`$nonEscape` перестанет экранировать специальные символы.

## Использование

```php
$nonEscape[message]
```

## Параметры

| Название  | Nbg    | Описание                                  | Нужно |
| --------- | ------ | ----------------------------------------- | ----- |
| сообщение | строка | Текст, который вы не хотите экранировать. | да    |

## Пример(ы)

Это перестанет экранировать определённые символы:

```javascript
bot.command({
    имя: 'nonEscape',
    код: `
  $nonEscape[Hello [;)]]
  `
});
```
