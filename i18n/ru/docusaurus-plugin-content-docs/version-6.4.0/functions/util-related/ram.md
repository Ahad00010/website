`$ram` возвращает использование ОЗУ.

## Использование

```php
$ram
```

## Пример(ы)

Это возвращает использование ОЗУ и доступное количество ОЗУ:

```javascript
bot.command({
    name: 'ram',
    code: `
  Я использую \`$ram%\` из \`$maxRam%\`!
  «
});
```
