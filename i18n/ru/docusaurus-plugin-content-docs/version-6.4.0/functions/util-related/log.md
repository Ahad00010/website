`$log` будет записывать данное сообщение в консоли вашего бота.

## Использование

```php
$log[content]
```

## Параметры

| Название | Nbg           | Описание                        | Нужно |
| -------- | ------------- | ------------------------------- |:-----:|
| контент  | строка, число | Содержимое сообщения в журнале. |  да   |

## Пример(ы)

Это будет записывать "Aoi.js отлично!" в консоли вашего бота:

```javascript
bot.command({
    name: "log",
    code: `
    $log[Aoi.js is great!]
    «
});
```
