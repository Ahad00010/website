`$spliceTextJoin`

## Использование

```php
$spliceTextJoin[text;splitter;sep1?;sep2?]
```

## Параметры

| Название    | Nbg    | Описание         | Нужно |
| ----------- | ------ | ---------------- |:-----:|
| текст       | строка | Текст для входа. |  да   |
| разделитель | строка | Как разделить.   |  да   |
| sep1        | строка | Разделитель.     | false |
| sep2        | строка | Разделитель.     | false |
