`$cloneChannel` клонирует канал.

## Использование

```php
$cloneChannel[channelID;name;returnID?]
```

## Параметры

| Название     | Nbg    | Описание                                                        | Нужно |
| ------------ | ------ | --------------------------------------------------------------- |:-----:|
| ID канала    | целое  | Идентификатор клонируемого канала.                              |  да   |
| название     | строка | Новое имя клонированного канала.                                |  да   |
| ID возврата? | строка | Если функция должна возвращать ID только что созданного канала. | ложь  |

**Он не клонирует ни одного сообщения этого канала.**

## Пример(ы)

Это клонирует текущий канал и назовет его "новый канал":

```javascript
bot.command({
    name: 'cloneChannel',
    code: `
  $cloneChannel[$channelID;new channel;false]
  `
});
```
