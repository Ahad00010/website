`$ownerID` + возвращает ID владельца сервера, в зависимости от заданного аргумента.

## Использование

```php
$ownerID[guildID?]
```

## Параметры

| Название    | Nbg   | Описание    | Нужно |
| ----------- | ----- | ----------- |:-----:|
| ID гильдии? | целое | ID гильдии. | ложь  |

## Пример(ы)

Это вернет ID владельца сервера, где вы выполняете команду в:

```javascript
bot.command({
    name: 'ownerID',
    code: `
  $ownerID[$guildID]
  `
});
```
