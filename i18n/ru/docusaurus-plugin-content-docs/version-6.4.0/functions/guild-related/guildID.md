`$guildID` вернет ID гильдии указанной гильдии.

## Использование

```php
$guildID[name?]
```

## Параметры

| Название | Nbg    | Описание                                | Нужно |
| -------- | ------ | --------------------------------------- |:-----:|
| имя?     | строка | Имя гильдии, которое вы хотите вернуть. | ложь  |

## Пример(ы)

Это вернет ваш ID гильдии:

```javascript
bot.command({
    name: 'guildID',
    code: `
  $guildID
  `
});
```
