`$oldPresence` содержит старые данные о присутствии пользователя, используемые в событии наличия обновлений или другие данные будут пустыми

## Использование

```php
$oldPresence[option]
```

## Параметры

| Название | Nbg    | Описание    | Нужно |
| -------- | ------ | ----------- |:-----:|
| опция    | строка | Извлечение. |  да   |
