`$newVariable` получает данные от событий переменных.

## Использование

```php
$newVariable[option;sep?]
```

## Параметры

| Название | Nbg    | Описание     | Нужно |
| -------- | ------ | ------------ |:-----:|
| опция    | строка | Извлечение.  |  да   |
| Sep      | строка | Разделитель. | ложь  |
