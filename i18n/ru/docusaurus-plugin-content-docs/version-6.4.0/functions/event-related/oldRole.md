`$oldRole` имеет данные о роли, прежде чем она была обновлена, используется в событии обновления ролей, иначе любые данные будут пустыми.

## Использование

```php
$oldRole[option]
```

## Параметры

| Название | Nbg    | Описание    | Нужно |
| -------- | ------ | ----------- |:-----:|
| опция    | строка | Извлечение. |  да   |
