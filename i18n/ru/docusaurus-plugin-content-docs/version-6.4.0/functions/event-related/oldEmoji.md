`$oldEmoji` держит данные для эмодзи до его обновления/удаления. (событие emojiUpdate и emojiDelete

## Использование

```php
$oldEmoji[option]
```

## Параметры

| Название | Nbg    | Описание    | Нужно |
| -------- | ------ | ----------- |:-----:|
| опция    | строка | Извлечение. |  да   |
