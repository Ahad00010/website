`$fetchClientValues` будет возвращать данные всех шардов клиента.

## Использование

```php
$fetchClientValues[function]
```

## Параметры

| Название | Nbg    | Описание                | Нужно |
| -------- | ------ | ----------------------- |:-----:|
| функция  | строка | Функция для выполнения. |  да   |
