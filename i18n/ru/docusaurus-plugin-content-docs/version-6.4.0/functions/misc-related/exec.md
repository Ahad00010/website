`$exec` выполнит данный код в вашей консоли.

## Использование

```php
$exec[code]
```

## Параметры

| Название | Nbg    | Описание                      | Нужно |
| -------- | ------ | ----------------------------- |:-----:|
| код      | строка | Код для выполнения в консоли. |  да   |

## Пример(ы)

Это вернёт текущую версию узла:

```javascript
bot.command({
    name: "exec",
    code: `
    \`\`\`
    $exec[node -v]
    \`\`\`
  `
});
```
