`$eventEmit` выдаст пользовательское событие.

## Использование

```php
$eventEmit[name;...datas]
```

## Параметры

| Название | Nbg    | Описание                           | Нужно |
| -------- | ------ | ---------------------------------- |:-----:|
| название | строка | Пользовательское название события. |  да   |
| ...datas | строка | Пользовательские данные событий.   |  да   |
