`$getCacheData` будет возвращать данные кэша.

## Использование

```php
$getCacheData[type;name;key;option?]
```

## Параметры

| Название | Nbg    | Описание | Нужно |
| -------- | ------ | -------- |:-----:|
| nbg      | строка |          |  да   |
| название | строка |          |  да   |
| ключ     | строка |          |  да   |
| опция?   | строка |          | false |
