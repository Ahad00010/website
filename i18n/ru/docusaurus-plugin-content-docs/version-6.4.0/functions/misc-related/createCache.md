`$createCache` создаст собственный кэш.

## Использование

```php
$createCache[type;name;options]
```

## Параметры

| Название | Nbg    | Описание           | Нужно |
| -------- | ------ | ------------------ |:-----:|
| nbg      | строка | Cache type.        |  да   |
| название | строка | Название кэша.     |  да   |
| опции    | строка | Опции кэширования. |  да   |
