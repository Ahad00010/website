`$userID` вернет ID данного пользователя.

## Использование

```php
$userID[user]
```

## Параметры

| Название     | Nbg           | Описание                                                         | Нужно |
| ------------ | ------------- | ---------------------------------------------------------------- |:-----:|
| пользователь | строка, число | Имя пользователя, из которого вы хотите вернуть ID пользователя. |  да   |

## Пример(ы)

Это вернет ваш идентификатор:

```javascript
bot.command({
    name: 'userID',
    code: `
  $userID[$username]
  `
});
```
