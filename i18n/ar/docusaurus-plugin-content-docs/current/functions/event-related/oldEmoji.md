`$oldEmoji` holds data for the emoji before it was updated/ deleted. (emojiUpdate and emojiDelete event)

## الاستخدام

```php
$oldEmoji[option]
```

## البارامترات

| Field  | النوع  | الديسكبربشين        | مطلوب |
| ------ | ------ | ------------------- |:-----:|
| option | string | Option to retrieve. | true  |
