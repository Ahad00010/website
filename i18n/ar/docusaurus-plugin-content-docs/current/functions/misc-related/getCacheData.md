`$getCacheData` will return cache data.

## الاستخدام

```php
$getCacheData[type;name;key;option?]
```

## البارامترات

| Field   | النوع  | الديسكبربشين | مطلوب |
| ------- | ------ | ------------ |:-----:|
| type    | string |              | true  |
| name    | string |              | true  |
| key     | string |              | true  |
| option? | string |              | false |
