`$spliceTextJoin`

## الاستخدام

```php
$spliceTextJoin[text;splitter;sep1?;sep2?]
```

## البارامترات

| Field    | النوع  | الديسكبربشين  | مطلوب |
| -------- | ------ | ------------- |:-----:|
| text     | string | Text to join. | true  |
| splitter | string | How to split. | true  |
| sep1     | string | Separator.    | false |
| sep2     | string | Separator.    | false |
