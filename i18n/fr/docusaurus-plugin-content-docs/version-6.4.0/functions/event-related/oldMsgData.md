`$oldMsgData` contient les données de l'ancien message.

## Utilisation

```php
$oldMsgData[option]
```

## Paramètres

| Champ  | Type                 | Description         | Obligatoire |
| ------ | -------------------- | ------------------- |:-----------:|
| option | chaîne de caractères | Option à récupérer. |    vrai     |
