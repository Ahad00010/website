`$newState` conserve les données de l'état vocal de l'utilisateur après la mise à jour (événement VoiceStateUpdate).

## Utilisation

```php
$newState[option]
```

## Paramètres

| Champ  | Type                 | Description         | Obligatoire |
| ------ | -------------------- | ------------------- |:-----------:|
| option | chaîne de caractères | Option à récupérer. |    vrai     |
