`$newPresence` conserve les données de mise à jour de la présence.

## Utilisation

```php
$newPresence[option]
```

## Paramètres

| Champ  | Type                 | Description         | Obligatoire |
| ------ | -------------------- | ------------------- |:-----------:|
| option | chaîne de caractères | Option à récupérer. |    vrai     |
