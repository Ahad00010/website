`$oldMessage` contient le contenu du message avant qu'il ne soit mis à jour. (s'il y en a)

## Utilisation

```php
$oldMessage[option]
```

## Paramètres

| Champ  | Type                 | Description         | Obligatoire |
| ------ | -------------------- | ------------------- |:-----------:|
| option | chaîne de caractères | Option à récupérer. |    vrai     |
