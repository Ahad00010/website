`$oldEmoji` conserve les données pour l'émoji avant sa mise à jour/suppression. (évènements emojiUpdate et emojiDelete)

## Utilisation

```php
$oldEmoji[option]
```

## Paramètres

| Champ  | Type                 | Description         | Obligatoire |
| ------ | -------------------- | ------------------- |:-----------:|
| option | chaîne de caractères | Option à récupérer. |    vrai     |
